버블 정렬 ( bubble sort ) 란 ?
: 두 인접한 데이터를 비교해서, 앞에 있는 데이터가 뒤에 있는 데이터보다 크면, 자리를 바꾸는 알고리즘
: swap이 없을때까지 계속 반복해서 정렬한다.

list의 요소가 n개 있는 경우
: (n-1) 턴이 이뤄진다.
: 만약 n번째 턴에 swap이 한번도 일어나지 않은 경우 중단 시킬 수 있다.
1번째 턴 - (n-1)번 조건체크 0~n-2까지 a[i], a[i+1] 체크
2번쩨 턴 - (n-2)번 조건체크 0~n-3까지
3번째 턴 - (n-3)번 조건체크 0~n-4까지
...

시간 복잡도 ?
: O(n^2)
: 최악의 경우 n*(n-1) / 2
: 최선의 경우 O(n) - 완전 정렬이 되어있는 경우
